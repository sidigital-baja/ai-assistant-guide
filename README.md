**Do I need to know how to code?**
No. Every command is shown exactly as you'll type it. Claude itself acts as your coding partner throughout the build — paste any section into Claude and it will write the code and walk you through the next step.

**How long will it take?**
The original build took several days of experimentation. With this guide — all errors already solved — most people complete a working version in a weekend. Phases 1–4 (first working version) can be done in a few hours.

**Can I build a different character?**
Absolutely. The guide covers persona possibilities extensively: Marilyn Monroe, The Terminator, James Bond, Yoda, or a completely original character. Only the personality details change — the technical steps are identical.

**Mac or Windows?**
Written primarily for Mac (Terminal + Homebrew). Most steps are identical on Windows with minor differences in Python setup. The deployed cloud version works on any platform.
