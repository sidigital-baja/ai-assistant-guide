01

AI Persona Design

Choose a character, define personality traits, speech patterns, and signature phrases. Build the system prompt that makes your assistant feel alive.

02

Custom Voice

Create a unique voice from a text description using ElevenLabs. Generate multiple variants, pick the best one, and wire it to your assistant.

03

Dev Environment

Install Python and VS Code, set up your API keys, and create the project file structure. Every command shown exactly as you'll type it.

04

Core Assistant

Build the brain: system prompt, persistent memory, intent detection, web search, and Claude API integration.

05

Web UI + GitHub

Build a browser-based interface with Streamlit, apply your custom visual theme, and connect voice input and output. Create your GitHub account here too — ready for deployment.

06

Google Workspace

Connect Gmail, Google Calendar, and Google Drive. Read emails, send with your signature, check your schedule, and search files.

07

Cloud Deployment

Push to GitHub and deploy on Streamlit Cloud. Your assistant goes live at its own URL, accessible from any device, anywhere.

08

Mobile App

Add your assistant to your iPhone or Android home screen as a full-screen progressive web app. One tap and you're in.
